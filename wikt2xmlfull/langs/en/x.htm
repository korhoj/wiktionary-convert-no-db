The following documentation is located at Module:languages/data3/x/documentation. [edit]

This module contains definitions and metadata for three-letter language codes starting with "x" on Wiktionary. See Wiktionary:Languages for more information.

This module must not be used directly in other modules or templates. The data should be accessed through Module:languages.
